### Project 9: React Countdown Game
- [ ] **Description:** A simple countdown game built with React, focusing on demonstrating the use of `useRef` along with `portal` and advanced `ref` techniques. The game starts a countdown from a specified time and updates the UI in real-time.
* Primary focus on `useRef` for accessing and manipulating DOM elements.
* Use of `portal` for rendering elements outside the main DOM hierarchy.
* Implementation of `forwardRef` and `useImperativeHandle` to pass `props` from the **parent component** to the **child component**.
- [ ] **Features:** Start and stop countdown, real-time countdown, and a reset feature.
- [ ] **GitHub Repository:** [Link](https://github.com/shanibider/React-Countdown-Game)
